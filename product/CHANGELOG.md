# Changelog

The released changelog for the `@kabaka/ai-dlc` package now lives at
[`installer/CHANGELOG.md`](installer/CHANGELOG.md). It is maintained
**automatically** by [release-please](https://github.com/googleapis/release-please):
each release PR updates that file (in [Keep a Changelog](https://keepachangelog.com/en/1.1.0/)
format) alongside the version bump in `installer/package.json`, so it stays
co-located with the package it describes. Do not hand-edit versions or release
sections there — add entries under its `## [Unreleased]` heading and let
release-please cut the release.
